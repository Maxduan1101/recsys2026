Use a budgeted global tail reranker: keep the high-nDCG head mostly fixed, then replace repeated/popular/BM25-collapse items in ranks 11‚Äì20 with relevance-gated, globally underused candidates. # Composite break-even: ndcg_loss < 0.2 * cd_gain.

heap = build_all_tail_swaps(
    rows=Q,
    positions=range(H+1, 21),

used_ndcg_loss = 0.0

while heap:
    swap = heap.pop_best_lazy_recomputed()

    q, p, old, new = swap.q, swap.p, swap.old, swap.new

    if new in lists[q]:
        continue

    delta_cd = ((freq[new] == 0) - (freq[old] == 1)) / catalog_size
    delta_ndcg = estimate_delta_ndcg(q, p, old, new)

    # Never spend diversity budget on a negative-CD swap unless it improves nDCG.
    if delta_cd < 0 and delta_ndcg <= 0:
        continue

    # Position-specific relevance/risk gate.
    if not passes_position_gate(q, p, old, new):
        continue

    # Full composite + MMR/xQuAD/frequency score.
    J = score_swap(q, p, old, new, lists[q], freq)

    if J <= margin_for_position[p]:
        continue

    ndcg_loss = max(0.0, -delta_ndcg)

    if used_ndcg_loss + ndcg_loss > ndcg_loss_budget:
        continue

    # Apply swap.
    lists[q][p-1] = new
    freq[old] -= 1
    if freq[old] == 0:
        del freq[old]
    freq[new] += 1
    used_ndcg_loss += ndcg_loss

    refresh_heap_entries_for_row_and_tracks(q, old, new)

Practical first submission recipe

Use this as your first conservative diversified blind run:

Lock ranks 1‚Äì10.

Only replace ranks 11‚Äì20.

Candidate pool: top 300‚Äì500 per row from hybrid retrieval/reranking.

Replace mostly tracks with freq[old] > 1.

Require new to be first-use globally unless it is a strong nDCG improvement.

Use xQuAD over artist, tag, mood/genre, and decade aspects.

Apply a soft penalty to BM25-only legacy/popular tracks in ranks 13‚Äì20.

Target catalog_diversity ‚âà 0.05.

Stop if estimated nDCG drops below about 0.190‚Äì0.191.

If dev/bootstrap estimates are stable, a second, more aggressive run can target 0.065‚Äì0.075. I would not target 0.10+ until the estimator shows that ranks 17‚Äì20 can absorb the swaps with less than roughly 0.01 nDCG loss.

The core principle is: do not diversify blindly; spend the lower top-20 positions on globally new, locally plausible, aspect-covering tracks, and price every swap in composite-score units before submitting.
